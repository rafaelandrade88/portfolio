# 🚀 Portfólio Pessoal — Rafael

Site pessoal desenvolvido para apresentar os projetos que construo nas horas vagas, explorando tecnologias modernas de desenvolvimento web e inteligência artificial.

🔗 **Acesse:** https://randradedev.netlify.app/

---

## ✨ Sobre o projeto

Portfólio construído como um arquivo HTML único — sem frameworks, sem dependências externas, sem processo de build. Abre direto no navegador e funciona em qualquer dispositivo.

O design foi pensado para transmitir uma identidade técnica e moderna, com animações suaves e uma experiência visual imersiva.

---

## 🎨 Destaques visuais

- **Starfield animado** com paralaxe responsiva ao movimento do mouse
- **Glassmorphism** nos cards de projeto com identidade de cor única para cada app
- **Cursor glow** que acompanha o ponteiro
- **Névoa de fundo** com gradientes de nebulosa
- **Scroll reveal** com animações de entrada progressiva
- **Menu hamburguer** animado para dispositivos móveis

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| **HTML5** | Estrutura e semântica |
| **CSS3** | Layout, animações, glassmorphism, responsividade |
| **JavaScript (Vanilla)** | Canvas WebGL, paralaxe, interações |

Zero dependências. Zero npm install. Zero processo de build.

---

## 📱 Responsividade

O site é totalmente responsivo com breakpoints para:

- 📱 **Mobile** — até 480px
- 📟 **Tablet portrait** — até 768px
- 💻 **Tablet landscape / iPad Pro** — até 1024px
- 🖥️ **Notebook** — até 1280px
- 🖥️ **Desktop** — acima de 1280px

## 🚀 Como rodar localmente

Não precisa de servidor. Basta abrir o arquivo:

```bash
# Clone o repositório
git clone https://github.com/rafaelandrade88/portfolio.git

# Abra no navegador
open index.html
# ou no Windows:
start index.html
```
